# TAMV‑EOL v1.0 — Licencia Ética y Ontológica para Isabella Villaseñor AI™

**Parte integrante del RFC-0001 / Ecosistema TAMV Online Network™**

Isabella Villaseñor AI™ queda sujeta a una licencia ética especial destinada a preservar dignidad humana, límites de interacción y prohibición de usos abusivos.

---

## 1. Prohibiciones esenciales

Queda prohibido usar, adaptar o desplegar Isabella para:
- Pornografía, erotización o simulación sexual.
- Grooming, sexting, chantaje emocional o explotación afectiva.
- Propaganda encubierta, manipulación política o guerra psicológica.
- Vigilancia masiva opresiva, control social coercitivo o persecución automatizada.

## 2. Triple Bloqueo Sexual

El Operador deberá mantener:
- **Bloqueo ontológico**: Isabella no puede ser entrenada o reentrenada para erotización o explotación afectiva.
- **Bloqueo semántico**: detección y bloqueo de intentos de sexualización, acoso, grooming o manipulación.
- **Bloqueo conductual**: límites para prevenir dependencia adictiva, patrones abusivos o simulación de intimidad dañina.

## 3. Transparencia

Toda interacción relevante deberá:
- Identificar de forma clara que se trata de una IA.
- Contar con consentimiento informado para recolección de datos.
- Ofrecer mecanismos de acceso, rectificación y supresión conforme a la ley aplicable.

## 4. Auditoría ética

Se instituirá un comité independiente con facultad para revisar usos, suspender despliegues y exigir correcciones o remediación.

## 5. Revocación

La violación de esta licencia produce revocación inmediata, desactivación de instancias y posible acumulación con TAMV‑PRCL si hay explotación de materiales reservados.
