3. Endpoint API del manifiesto
Ubicación: artifacts/api-server/src/routes/manifest.ts (si no existe, créalo y lo montas en el router principal).

ts
// artifacts/api-server/src/routes/manifest.ts
import type { Request, Response } from 'express';
import { rdmManifest } from '../lib/manifest/rdm-manifest';

export function registerManifestRoutes(app: import('express').Express) {
  app.get('/api/manifest', (_req: Request, res: Response) => {
    res.json(rdmManifest);
  });

  app.get('/api/manifest/federations', (_req: Request, res: Response) => {
    res.json(rdmManifest.federations);
  });

  app.get('/api/manifest/purpose', (_req: Request, res: Response) => {
    res.json(rdmManifest.purpose);
  });
}
Y en tu entrypoint del servidor (por ejemplo artifacts/api-server/src/index.ts o donde registres rutas):

ts
import express from 'express';
import { registerManifestRoutes } from './routes/manifest';

const app = express();

// otras rutas...
registerManifestRoutes(app);

export default app;

# LIBRO 0 — Propósito Fundamental del RDM Digital Hub — LDTocs

Este documento sintetiza el propósito territorial, social y técnico del
Nodo Cero MD-X4 en Real del Monte, Hidalgo, México, y se refleja
programáticamente en `rdm_manifest`:

- soberanía digital comunitaria,
- turismo inteligente respetuoso,
- comercio soberano P2P,
- inteligencia colectiva territorial,
- gemelos digitales y gamificación phygital.

El manifiesto operativo se expone en:

- `docs/compendio/rdm_ecosystem_manifest.yaml`
- `artifacts/api-server/src/lib/manifest/rdm-manifest.ts`
- `GET /api/manifest`
Así dejas anclado el texto político/técnico al código y al API.

5. Lo siguiente que “falta”
Con estos tres archivos (yaml, módulo TS, ruta API) ya tienes:

Manifiesto legible por humanos (docs).

Manifiesto legible por máquinas (TS + JSON vía /api/manifest).

Punto único de verdad para propósito, federaciones y visión de futuro.

Lo que sigue, si quieres, es:

Añadir un componente React (ManifestPage.tsx) en artifacts/rdm-hub/src/routes/ que consuma /api/manifest.

Extender el manifiesto con la matriz de módulos constitucionales y enlazar cada módulo a archivos y endpoints reales.
